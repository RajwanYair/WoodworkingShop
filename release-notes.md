## What's New in v2.5.0

### New Furniture Types
- **Desk** — desktop, side panels (legs), modesty panel, optional under-desk shelves
- **Wardrobe** — hanging rail, doors, adjustable shelves, tall carcass support

### New Features
- **Custom Materials Editor** — add/edit/remove custom panel & back materials (persisted in localStorage)
- **Shaker Door Style** — flat, shaker, or no-door options for cabinets & wardrobes
- **SVG Export** — download any preview view as an SVG file
- **G-code Export** — CNC router G-code for cut sheets (per-sheet & combined download)
- **Help / Onboarding Overlay** — 5-step guided walkthrough for new users

### Accessibility
- Focus trap in onboarding modal & sidebar overlay
- Escape key dismisses modals and overlays
- aria-labels, aria-expanded, role="tablist"/role="tab" across all interactive elements

### Testing
- 197 tests across 18 files covering desk/wardrobe parts & assembly, G-code export, custom materials store, and more
